# INDUSTRIAL FSM WORK ORDER FIELD SPECIFICATION

## Legend

| Marker         | Meaning                           |
| -------------- | --------------------------------- |
| **[REQ]**      | Required / Mandatory Field        |
| **[OPT]**      | Optional Field                    |
| **[COND]**     | Conditionally Required            |
| **[AUTO]**     | System Generated / Auto-populated |
| **[READONLY]** | Non-editable by field technicians |

---

# 1. WORK ORDER HEADER

| Field                       | Requirement           | Description                                      |
| --------------------------- | --------------------- | ------------------------------------------------ |
| Work Order ID               | [REQ][AUTO][READONLY] | Unique identifier generated by FSM/ERP           |
| Parent Work Order           | [OPT]                 | Linked parent maintenance package                |
| Service Request ID          | [OPT]                 | Associated customer/service request              |
| Work Type                   | [REQ]                 | Preventive / Corrective / Emergency / Inspection |
| Maintenance Strategy        | [OPT]                 | CBM, Predictive, Preventive, Reactive            |
| Priority Level              | [REQ]                 | Low / Medium / High / Critical                   |
| Risk Classification         | [OPT]                 | Operational risk category                        |
| Status                      | [REQ][AUTO]           | Current workflow status                          |
| Workflow Stage              | [AUTO][READONLY]      | FSM lifecycle stage                              |
| Created Date                | [REQ][AUTO][READONLY] | Timestamp of creation                            |
| Scheduled Start             | [REQ]                 | Planned field execution start                    |
| Scheduled Finish            | [REQ]                 | Planned completion time                          |
| SLA Target                  | [COND]                | Mandatory if SLA-enabled customer                |
| Estimated Labor Hours       | [OPT]                 | Planning estimate                                |
| Site Region                 | [OPT]                 | Regional operational zone                        |
| Site Location               | [REQ]                 | Physical service location                        |
| GPS Coordinates             | [COND][AUTO]          | Required for mobile workforce tracking           |
| Business Unit               | [REQ]                 | Owning operational department                    |
| Department                  | [REQ]                 | Responsible maintenance division                 |
| Cost Center                 | [COND]                | Required for enterprise billing/accounting       |
| Permit Requirement          | [REQ]                 | Indicates permit dependency                      |
| Confined Space Entry        | [REQ]                 | Safety classification                            |
| Hot Work Required           | [REQ]                 | Welding/cutting/fire-producing activity          |
| Lockout/Tagout Required     | [REQ]                 | Energy isolation requirement                     |
| Environmental Sensitivity   | [OPT]                 | Environmental impact level                       |
| Regulatory Compliance Scope | [OPT]                 | Applicable standards                             |
| Escalation Tier             | [OPT]                 | Incident escalation severity                     |
